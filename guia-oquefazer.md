# Oquefazer — Guia de Operação

---

## 1. Subir código no GitHub

1. Abre **github.com/joaubaron/oquefazer**
2. Clica em **index.html** → ícone ✏️
3. Apaga tudo e cola o novo código
4. Clica em **Commit changes** duas vezes

✅ Site atualiza em menos de 1 minuto.

---

## 2. Trocar anunciante no código

### Parceiro do Mês
Busca no HTML o comentário `PARCEIRO DO MÊS` e troca:

| campo | o que colocar |
|---|---|
| `1️⃣ href` | link do Instagram |
| `2️⃣ anunciante_id` | `mensal_ano_mes_nomeparceiro` |
| `3️⃣ parceiro-nome` | nome do negócio |
| `4️⃣ parceiro-label` | frase curta + emoji |
| `5️⃣ parceiro-acesse` | horário de funcionamento |
| `6️⃣ parceiro-cidade` | cidade do anunciante |
| `7️⃣ logo_anunciante.png` | subir a logo no GitHub |

> ⚠️ A cidade do parceiro precisa bater com a cidade que o usuário buscar — só assim o destaque aparece.

### Parceiros da Semana
Busca `SEMANA 1 · ESQUERDA / MEIO / DIREITA` e troca:

| campo | o que colocar |
|---|---|
| `1️⃣ href` | link do Instagram |
| `2️⃣ anunciante_id` | `semanal_ano_mes_sX_pos_nome` |
| `3️⃣ semana-nome` | nome do negócio |
| `4️⃣ semana-emoji` | emoji temático |
